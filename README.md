# Nu_flow
A distraction free terminal writing app in the tradition of wordgrinder

A simple drafting tool with the ability to change the font size.

Made with the assistance of claude code

The main app is made to work wit hthe kitty terminal emulator- I included the kitty config file that you may or may not need to add to get this to work- the tty version of the app work in the proper terminal, the difference is that the text does not fade out and you cannot chnge the font size.
